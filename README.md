## 项目概述 java245


## 技术栈

- **前端框架**：Vue3
- **组件库**：Element Plus
- **构建工具**：Vite
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **数据可视化**：ECharts

## 功能模块

系统包含以下页面：

1. 登录页
2. 注册页
3. 仪表盘首页
4. 水果分类管理
5. 水果库存管理
6. 订单管理
7. 用户管理
8. 角色管理
9. 权限管理
10. 个人中心
11. 系统设置
12. 错误页面

### 自定义组件

- 共包含 15 个自定义组件

### 特性

- 分页功能
- 使用 Pinia 实现状态管理
- ECharts 数据可视化
- 父子组件通信（props/emits）
- Vue 指令（v-if/v-for/v-on/v-model/v-bind）
- 计算属性（computed）与侦听器（watch）
- RBAC 权限控制系统

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 现代浏览器

## 目录结构

```
src/
├── assets/             # 资源文件夹
├── components/         # 组件目录
├── views/              # 页面目录
├── router/             # 路由配置
├── store/              # 状态管理
├── utils/              # 工具方法
├── App.vue             # 主应用组件
└── main.js             # 入口文件
```

## 核心亮点

- 纯前端实现，无需后端数据库，便于快速部署和开发
- 页面美观，符合业务需求，功能完整
- 支持定制开发，提供开发文档，方便学习和二次开发
- 实现了基于角色的权限控制，保障系统安全
